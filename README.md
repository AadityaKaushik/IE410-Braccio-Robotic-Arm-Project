# IE410 Robotics Project – Part A

## Object Manipulation Using Robotic Arms

This repository contains the complete implementation, source code, documentation, experimental results, images, and demonstration videos for **Part A** of the IE410 Introduction to Robotics project.

The project focuses on robotic object manipulation using robotic arms through both pre-programmed and vision-assisted approaches.

---

## Project Tasks

### Task 1: Pre-Programmed Pick and Place

Implementation of a basic pick-and-place operation using predefined robotic arm movements.

### Task 2: Camera-Assisted Pick and Place

Integration of computer vision techniques to detect and manipulate objects using camera feedback.

### Task 3: Object Handover Between Two Robot Arms

Coordination and synchronization between two robotic arms for object transfer operations.
